## Travel Orchestrator Example

Run the first dependency-free orchestrator script:

```bash
python3 travel_orchestrator.py \
  --origin Dublin \
  --destination Paris \
  --start-date 2026-09-10 \
  --end-date 2026-09-17 \
  --travelers 2 \
  --budget mid-range \
  --interests food,museums,walking
```
